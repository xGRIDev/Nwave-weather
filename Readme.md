Nwave-Weather Project are Based on Case [Roadmap-Golang](https://roadmap.sh/projects/weather-api-wrapper-service/) which using couple tools like (Redis, API-Weather, Etc).

# Tech-Stack
- [Golang](https://go.dev/) (V.1.29+)
- [Redis](https://redis.io/)
- [OpenWeather](https://openweathermap.org/api) as 3rd-Party API

# Framework
- [GIN](https://pkg.go.dev/github.com/gin-gonic/gin) (HTTP Framework)
- [Godotenv](https://github.com/joho/godotenv) (Configruation)
- [Redis](https://pkg.go.dev/github.com/gin-gonic/gin) (Req & Respond)
